이용약관

OneMouse는 Windows PC와 Android 기기를 로컬로 연결해 입력 제어와 파일·클립보드 전송을 돕는 도구입니다. 아래 약관은 앱과 웹사이트 사용 조건을 설명합니다.

시행일: 2026년 5월 6일

1. 서비스 목적

OneMouse는 사용자가 소유하거나 사용 권한이 있는 PC와 Android 기기를 연결하는 로컬 생산성 도구입니다. 타인의 기기 또는 허가받지 않은 네트워크에서 사용하는 것은 허용되지 않습니다.

2. 사용자의 책임

  • 사용자는 접근성 권한, 네트워크 권한, 파일 접근 권한을 직접 확인하고 허용해야 합니다.
  • 전송하는 파일과 클립보드 내용에 대한 책임은 사용자에게 있습니다.
  • 회사망, 공용망, 보안 정책이 있는 환경에서는 관리자 정책을 따라야 합니다.

3. FREE / PRO

현재 버전은 무료로 공개됩니다. 향후 PRO 기능은 Google Play 인앱결제로 전환될 수 있습니다. 기본 연결과 입력 제어는 무료에서도 유지하고, 여러 기기 저장, 기기별 위치 고정, 자동 재연결 같은 고급 편의 기능을 PRO로 제공하는 방향입니다.

4. 지원 환경

OneMouse는 Windows PC와 Android 기기가 같은 Wi-Fi 또는 같은 로컬 네트워크에 있는 환경을 기준으로 동작합니다. 게스트 Wi-Fi, AP 격리, 회사망 보안 정책처럼 로컬 통신이 제한된 환경에서는 연결이 되지 않을 수 있습니다.

5. 보증의 제한

OneMouse는 네트워크, 운영체제 권한, 보안 프로그램, 기기 제조사 정책의 영향을 받을 수 있습니다. 모든 환경에서 중단 없는 연결 또는 파일 전송을 보장하지 않습니다.

6. 약관 변경

기능, 결제 방식, 배포 정책이 변경되면 본 약관도 갱신될 수 있습니다. 중요한 변경 사항은 웹사이트 또는 앱 배포 페이지를 통해 안내합니다.
