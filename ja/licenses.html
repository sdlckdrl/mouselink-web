オープンソースライセンス

OneMouse は PC アプリ、Android アプリ、QR スキャン、決済、広告のためにオープンソースライブラリとサードパーティ SDK を使用します。

Android

Android アプリの実装には AndroidX と Kotlin ライブラリを使用します。QR スキャンには ZXing ベースのコンポーネントを使用します。

Google SDK

Pro の購入と復元には Google Play Billing Library を使用します。Free モードのバナー広告には Google Mobile Ads SDK を使用します。

PC アプリ

Windows アプリはデスクトップ UI、ローカル discovery、入力キャプチャ、転送機能のために Electron と Python コンポーネントを使用します。
