Technical & Security

接続構造、権限、データ処理を分けて説明します。

このページは日常的な操作手順ではなく、OneMouse が Windows PC と Android をどのように接続し、 どの権限を何のために使うのかを説明するページです。

データの流れ 権限 ネットワーク 対応範囲

Local First

OneMouse はユーザーがペアリングした端末間で動作します。

基本接続は、同じ Wi-Fi または同じルーター配下の Windows PC と Android 端末の間で行われます。 入力内容、ファイル、クリップボード内容を OneMouse のサーバーへアップロードする構造ではありません。

1
端末検出

PC アプリと Android アプリはローカルネットワーク内で互いを探します。ネットワーク設定によって検出が遅れる場合があります。

2
ユーザー確認

PC アプリに表示される QR コードまたは 6 桁コードで、ユーザーがペアリングを確認します。

3
入力制御

PC のマウス移動、クリック、スクロール、キーボード入力は、ペアリング済み Android へ制御イベントとして送られます。

4
ファイルとテキスト

ユーザーが選択したファイル、テキスト、クリップボード内容だけがペアリング済み端末間で移動します。

Permissions

権限は必要な機能のために使用します。

OneMouse は起動時にすべての権限を要求しません。関連する機能を使うときに必要な項目を案内します。

主要機能

アクセシビリティサービス

Android の仮想カーソル表示、クリック、スクロール、ドラッグ、Back、Home、キーボード入力など、ユーザーが要求した操作を実行するために使用します。

接続維持

フォアグラウンドサービス

PC と Android のローカル接続を維持し、実行中であることを通知で表示するために使用します。

端末検出

付近の Wi-Fi デバイス

Android 13 以降で同じネットワーク上の PC を探すために必要になる場合があります。位置追跡目的では使用しません。

状態表示

通知

接続サービスの実行状態やファイル転送状態を知らせるために使用します。

任意機能

システム設定の変更

PC のショートカットで Android の画面明るさを調整するときだけ使用します。

安定性

バッテリー制限の除外

画面オフ後に接続が頻繁に切れる場合のみ推奨します。

Data Handling

入力、ファイル、クリップボードは選択された操作の範囲で処理します。

OneMouse はユーザー自身のペアリング済み端末をつなぐためのツールです。ユーザーのファイルを保存する OneMouse アカウントサーバーは必要ありません。

  • マウスとキーボード入力は Android 制御イベントとして送られます。
  • ファイル転送はユーザーが選択またはドロップしたファイルに対してのみ開始されます。
  • クリップボードとテキスト転送は、ユーザーが機能または共有操作を使ったときだけ処理されます。
  • 広告分析のためにアクセシビリティデータを使用しません。

Review Notes

ユーザー案内とアプリ審査で重要な点

  • Windows PC アプリと Android アプリの両方が必要です。
  • アクセシビリティは Android 入力制御のための主要機能です。
  • フォアグラウンドサービスはユーザーが開始した接続と転送状態のために使用します。
  • PC と Android は同じ Wi-Fi または同じルーター配下にある必要があります。

Network

接続品質はローカルネットワークの条件に影響されます。

OneMouse はインターネットサーバー経由で制御を中継する製品ではありません。ペアリング済み端末同士がローカルネットワークで通信できる必要があります。

推奨環境

個人 Wi-Fi または同じルーター

  • PC と Android が同じルーターに接続されています。
  • ゲスト Wi-Fi と AP 分離が無効です。
  • PC ファイアウォールが OneMouse の通信を許可しています。
  • セキュリティソフトがローカル通信をブロックしていません。

注意環境

会社ネットワーク、公共 Wi-Fi、分離ネットワーク

  • 会社のセキュリティポリシーが端末検出をブロックする場合があります。
  • 公共 Wi-Fi は端末間通信を遮断する場合があります。
  • VPN やファイアウォール設定により PC が見えない場合があります。
  • 不安定な場合は、まず個人ルーター環境で確認してください。

Support Scope

公開ガイドは現在 Wi-Fi/LAN 接続を中心にしています。

一般ユーザーは Windows PC と Android を同じ Wi-Fi または同じローカルルーターで接続します。USB/ADB 接続は現在の公開サポート範囲に含まれません。
