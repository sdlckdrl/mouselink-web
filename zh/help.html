Help

按电源、权限、发现和位置缩小问题范围。

大多数问题来自应用电源、Android 权限、本地网络发现、配对或位置设置。

Power

检查应用电源

如果 Android 电源按钮关闭，后台工作和输入接收会停止。

Permissions

先检查无障碍权限

Android 输入控制需要 OneMouse 无障碍服务。

Battery

检查电池限制

如果息屏后断开，请从应用设置打开电池优化设置。

Discovery

检查 mDNS 和 UDP

访客 Wi-Fi、AP 隔离、公司网络和防火墙可能会阻止发现。

Layout

位置不正确时

旋转或显示变化后，重新保存手机位置。

Return

光标返回快捷键

如果光标没有从 Android 返回，请在 PC 上按 Ctrl + Alt + Backspace。

Troubleshooting

Follow this order.

Finding the failing step is faster than changing several settings at once.

1
Check app and permission state

Confirm PC app, Android power, foreground service, and Accessibility permission.

2
Check whether the PC is visible

Confirm both devices are on the same local network and run device search again.

3
Check pairing state

If pairing is required, use QR or the 6-digit code again.

4
Check input and placement

Check selected PC, phone position, entry direction, and rotation state.

5
Use recovery shortcut

Use Ctrl + Alt + Backspace if the cursor feels stuck on Android.
