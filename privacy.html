개인정보처리방침

OneMouse는 Windows PC와 Android 기기를 같은 로컬 환경에서 연결하기 위한 앱입니다. 본 방침은 앱과 웹사이트가 어떤 정보를 처리하고, 어떤 정보를 외부로 전송하지 않는지 설명합니다.

시행일: 2026년 5월 6일

1. 기본 원칙

OneMouse의 기본 연결 구조는 사용자의 PC와 Android 기기 사이의 로컬 통신입니다. 별도의 OneMouse 계정 서버를 통해 사용자의 파일, 클립보드 내용, 입력 내용을 수집하거나 저장하지 않습니다.

2. 처리하는 정보

  • 기기 식별 정보: Android 기기 ID, 기기 이름, PC 이름 또는 로컬 네트워크 주소
  • 연결 정보: 페어링 상태, 세션 키, 선택된 기본 PC, 마지막 연결 시간
  • 화면 배치 정보: Android 화면 크기, 방향, PC 모니터 주변의 폰 위치 설정
  • 전송 정보: 사용자가 직접 보낸 파일, 텍스트, 클립보드 내용
  • 결제 정보: PRO 구매 여부와 구매 복원 결과. 실제 결제 처리는 Google Play가 담당합니다.

3. 접근성 권한 사용

Android 접근성 서비스는 PC에서 전달된 마우스·키보드 입력을 Android 화면에 적용하기 위해 사용됩니다. OneMouse는 접근성 권한을 광고 추적, 화면 내용 수집, 비밀번호 수집, 원격 서버 전송 목적으로 사용하지 않습니다.

4. 파일과 클립보드

파일과 클립보드는 사용자가 직접 전송하거나 동기화를 켠 경우에만 PC와 Android 사이에서 이동합니다. 전송된 파일은 사용자의 기기 저장 위치에 저장되며, OneMouse 운영 서버에 업로드되지 않습니다.

5. 보관과 삭제

페어링 정보, 세션 키, 기기 목록, 위치 설정은 앱 내부 저장소 또는 PC 설정 파일에 저장될 수 있습니다. 사용자는 앱의 기기 삭제, 앱 데이터 삭제, 앱 제거를 통해 저장된 정보를 삭제할 수 있습니다.

6. 제3자 제공

OneMouse는 사용자의 파일, 클립보드, 입력 내용을 제3자에게 판매하거나 제공하지 않습니다. 단, 인앱결제 과정에서는 Google Play 결제 시스템이 구매 처리를 담당하며 Google의 약관과 정책이 적용됩니다.

7. 문의

개인정보 관련 문의는 앱 배포 페이지에 표시된 개발자 연락처로 접수합니다. Play Console 등록 전 실제 개발자 이메일을 이 항목에 반영해야 합니다.
