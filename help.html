Help

권한, 발견, 복귀 문제를 빠르게 좁힙니다.

문제 원인은 권한, 네트워크 discovery, 페어링, 배치로 나눠 보는 편이 빠릅니다. 아래 순서대로 확인하면 대부분의 막힘을 좁힐 수 있습니다.

Permissions

권한이 먼저입니다

Android 입력 제어는 접근성 서비스가 켜져 있어야 합니다. 권한이 꺼져 있으면 PC가 보여도 실제 입력은 들어가지 않습니다.

Pairing

QR 또는 6자리 코드

PC 하단 QR을 스캔하거나 6자리 코드를 입력합니다. 페어링 전에는 연결 버튼이 활성화되지 않는 것이 정상입니다.

Discovery

mDNS와 UDP를 확인합니다

회사망, 게스트 Wi-Fi, AP 격리 환경에서는 mDNS와 UDP가 모두 막힐 수 있습니다. 같은 공유기 아래인지 먼저 확인하세요.

Firewall

방화벽과 네트워크를 확인합니다

PC가 보이지 않으면 같은 LAN인지, 게스트 Wi-Fi나 AP 격리 정책이 켜져 있지 않은지 확인합니다.

Layout

배치가 어긋날 때

Android 방향 전환 후에는 화면 크기와 위치가 다시 동기화됩니다. 그래도 어색하면 폰 위치를 다시 저장해 주세요.

Return

커서 복귀 단축키

PC로 커서가 돌아오지 않으면 Ctrl + Alt + Backspace를 눌러 capture를 해제하고 복귀 메시지를 다시 보냅니다.

Troubleshooting

문제 해결 순서

어느 단계에서 막히는지 먼저 나누면 해결이 훨씬 빨라집니다.

1
앱과 권한 상태 확인

PC 앱, Android 앱, 포그라운드 서비스, 접근성 권한이 모두 살아 있는지 확인합니다.

2
PC가 보이는지 확인

mDNS/UDP discovery가 가능한 같은 로컬 네트워크인지 확인합니다.

3
페어링 상태 확인

PC 목록에 페어링 필요로 보이면 QR 또는 6자리 코드를 다시 사용합니다. 이미 꼬였으면 unpair 후 다시 연결합니다.

4
입력과 배치 확인

PC에서 선택된 기기, Android 위치, 진입 방향, 화면 방향 변경 후 동기화 상태를 확인합니다.

5
복구 단축키 사용

커서가 Android 쪽에 갇힌 느낌이면 Ctrl + Alt + Backspace로 PC 입력을 강제 복귀합니다.

Transfer

파일과 클립보드가 안 될 때

  • PC에서 어떤 Android가 선택되어 있는지 확인합니다. PC -> Android 파일은 선택된 기기로 갑니다.
  • 텍스트 선택 메뉴에서 OneMouse to PC를 누르거나 Android 공유 메뉴에서 OneMouse를 고르면 기본 PC로 보냅니다.
  • OneMouse to PC가 보이지 않으면 더보기의 앱 관리에서 One Mouse를 켭니다. 이 설정은 보통 처음 한 번만 필요합니다.
  • 대용량 파일은 네트워크 상태에 영향을 받으므로 연결이 안정적인지 먼저 확인합니다.
  • 클립보드는 파일 전송 TCP 채널을 재사용하므로 pairing과 세션 키 상태가 중요합니다.

FAQ

자주 묻는 질문

최신 연결 구조를 기준으로 답변을 다시 정리했습니다.

중간 서버를 거치나요?
기본 구조는 같은 로컬 네트워크 안에서 PC와 Android가 직접 통신하는 방식입니다.
mDNS가 있으면 UDP discovery는 없어졌나요?
아닙니다. mDNS를 우선 사용하지만 라우터나 네트워크 정책에 따라 막힐 수 있어 UDP discovery fallback을 유지합니다.
QR 스캔과 6자리 코드는 무엇이 다른가요?
둘 다 같은 페어링을 시작하는 방법입니다. QR은 입력을 줄이고, 6자리 코드는 카메라를 쓰기 어려울 때 사용할 수 있습니다.
여러 Android를 동시에 붙일 수 있나요?
가능합니다. PC에서 여러 기기를 발견하고 선택, 연결, 배치할 수 있습니다.
진입 경계 표시 토글은 무엇인가요?
PC 실제 모니터 가장자리에 테스트용 형광 진입선을 표시하는 설정입니다. 기본값은 꺼짐이며 앱 내부 배치 안내와는 별개입니다.
커서가 Android에서 PC로 안 돌아오면 어떻게 하나요?
PC에서 Ctrl + Alt + Backspace를 누르세요. PC capture를 해제하고 Android에 return_pc 메시지를 다시 보내는 복구 경로입니다.
Android에서 PC로 텍스트나 파일을 보낼 수 있나요?
Android 공유 메뉴에서 OneMouse를 선택하거나 클립보드 동기화를 사용하면 기본 PC로 텍스트와 파일을 보낼 수 있습니다.
