소개 사용 가이드 도움말 다운로드

Guide

처음 연결할 때 필요한 순서를 한 번에 정리했습니다.

설치 직후 어디서부터 시작해야 하는지 헷갈리지 않도록, 실제 연결 순서와 배치 팁을 짧고 선명하게 정리했습니다.

1
PC 앱 실행

Windows에서 OneMouse를 실행하고, 기기 검색이 시작된 상태인지 확인합니다.

2
Android 앱 열기

모바일에서 앱을 열고 연결 대기 상태로 둡니다. 권한 요청이 보이면 먼저 허용합니다.

3
6자리 페어링 코드 확인

PC에 표시된 코드를 모바일에 입력해 같은 기기인지 확인하고, 안전하게 페어링합니다.

4
모바일 위치 배치

모니터의 어느 가장자리로 진입할지 정하고, 모바일을 왼쪽·오른쪽·위·아래에 배치합니다.

5
마우스와 키보드 사용 시작

배치된 방향으로 커서를 넘기면 제어가 이동하고, PC 키보드로 모바일 입력도 이어집니다.

Checklist

빠른 시작 체크리스트

  • PC와 Android가 같은 Wi-Fi 또는 같은 로컬 네트워크에 연결되어 있는지 확인
  • PC 앱이 실행 중이고 모바일 앱이 연결 대기 상태인지 확인
  • 처음 연결이라면 6자리 코드 입력이 완료되었는지 확인
  • Android 쪽 접근성 또는 입력 관련 권한이 허용되었는지 확인
  • 모바일 위치가 실제 마우스 진입 방향과 맞게 배치되었는지 확인

USB Mode

USB 연결은 고급 대체 모드입니다

  • 회사망, 공용 Wi-Fi, 방화벽 때문에 Wi-Fi 검색이 어려울 때 사용하는 보조 연결 방식입니다.
  • Android에서 개발자 옵션과 USB 디버깅을 켜고, 최초 연결 시 PC의 RSA 키를 허용해야 합니다.
  • Windows 배포판에는 ADB 구성요소가 포함되어 있어 별도 Android Studio 설치 없이 사용할 수 있습니다.
  • USB 테더링이 아니라 ADB 터널링 방식이므로, `adb devices`에서 기기가 `device` 상태로 보여야 합니다.

Layout

듀얼 모니터와 여러 모바일을 쓸 때

  • 배치 화면에는 PC 모니터 구성이 그대로 축소되어 표시됩니다.
  • 여러 Android가 연결돼 있으면 각 기기를 겹치지 않게 외곽에 배치하는 것이 좋습니다.
  • 모바일에서는 다른 기기도 보이지만, 자신의 기기만 직접 이동할 수 있습니다.
  • PC에서는 선택된 기기가 파일 전송 대상이 되므로 현재 선택 상태를 함께 확인합니다.

Flow

실제 사용 흐름에 잘 맞는 순간

  • 테스트 기기 두세 대를 번갈아 조작하면서 앱 상태를 확인할 때
  • 모바일 검색창이나 로그인 입력을 PC 키보드로 빠르게 처리할 때
  • 스크린샷, APK, 문서 파일을 선택된 기기로 바로 보내야 할 때
  • 작업 중 메시지, 인증 코드, 메모를 확인하고 다시 PC 작업으로 돌아올 때
