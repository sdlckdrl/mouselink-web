Updated Guide

처음 연결부터 자주 쓰는 기능까지 순서대로 봅니다.

OneMouse는 PC 마우스와 키보드로 Android와 다른 Windows PC를 제어하도록 돕습니다. 이 가이드는 설치 후 바로 필요한 연결 순서, 앱 화면에서 볼 위치, 파일 전송, 모바일 화면 원격 제어, 단축키, 다른 PC 연결 흐름만 정리했습니다.

5분 연결 순서 원격 제어 필요한 권한 보기 다른 PC 연결 단축키 보기 기술/보안 안내

Quick Start

처음 연결은 이 순서만 따라가면 됩니다.

복잡한 네트워크 설정을 직접 만질 필요가 없습니다. PC 앱을 켜고 Android에서 PC를 찾은 뒤 QR 또는 6자리 코드로 확인합니다.

1
PC 앱을 실행합니다

Windows에서 OneMouse를 켜면 PC 화면에 6자리 코드와 QR 코드가 표시됩니다. 최신 Setup 파일은 다운로드 페이지에서 자동으로 연결됩니다.

2
Android에서 접근성 권한을 켭니다

PC 마우스와 키보드로 Android를 제어하려면 OneMouse 접근성 서비스가 필요합니다. 처음 설정에서 사실상 필수로 봐야 하는 항목은 이것 하나입니다.

3
PC를 검색합니다

PC와 Android가 같은 Wi-Fi 또는 같은 공유기 아래에 있으면 앱에서 PC를 찾을 수 있습니다. Android 13 이상에서는 이 순간에만 주변 Wi-Fi 기기 권한을 요청할 수 있습니다.

4
QR 또는 6자리 코드로 페어링합니다

PC 화면의 QR을 스캔하거나 6자리 코드를 입력합니다. 이 단계가 끝나야 입력, 파일, 클립보드 채널을 사용할 수 있습니다.

5
폰 위치를 PC 가장자리에 배치합니다

모니터 왼쪽, 오른쪽, 위, 아래 중 실제로 커서를 넘길 위치에 Android를 배치합니다. 이 배치가 커서 진입 방향이 됩니다.

6
입력, 전송, 모바일 화면 원격 제어를 사용합니다

마우스를 배치 방향으로 밀면 Android 또는 서브 PC로 넘어갑니다. PC 파일은 빠른 드롭존으로 보내거나, PC-to-PC에서는 파일을 잡은 채 화면 가장자리로 넘겨 대상 PC에서 놓을 수 있습니다. 기기 관리의 원격 제어 아이콘으로 모바일 화면 원격 제어 창을 열고, Print Screen 또는 Ctrl + Alt + S로 모바일 스크린샷을 PC 기본 뷰어에 띄울 수 있습니다.

페어링 순서 이미지로 보기 PC 코드 확인, 숫자 입력, QR 스캔 흐름을 한 장으로 확인합니다.

Position

연결 후에는 폰 위치만 실제 방향에 맞추면 됩니다.

마우스를 어느 모니터 가장자리로 넘길지 정하는 단계입니다. Android를 PC의 왼쪽, 오른쪽, 위, 아래 중 실제 사용할 방향에 배치하세요.

폰 위치 배치 화면 보기 커서가 넘어갈 방향을 정하는 PC 화면입니다.

Remote Control

모바일 화면 원격 제어는 기기 관리에서 시작합니다.

페어링된 Android 행의 원격 제어 아이콘을 누르면 별도 창이 열리고 Android 화면이 PC에 표시됩니다. 처음 시작할 때는 Android에서 화면 캡처 권한을 승인해야 하며, 승인 후 같은 창에서 바로 조작할 수 있습니다.

Controls

원격 제어 창에서 바로 조작합니다.

  • 왼쪽 클릭, 드래그, 휠 스크롤은 Android 화면에 그대로 전달됩니다.
  • 오른쪽 클릭은 뒤로가기, 가운데 클릭은 홈으로 동작합니다.
  • 한글을 포함한 PC 키보드 입력은 Android의 현재 입력 칸으로 전달됩니다.
  • 화면이 뜨지 않으면 Android 화면 캡처 승인 팝업이 앞에 떠 있는지 확인하고, 필요하면 중지 후 다시 시작합니다.
모바일 원격 제어 창 예시 보기 Android 화면을 PC 기본 창 안에서 바로 보고 조작하는 예시입니다.

Permissions

사용자가 켜야 하는 항목은 기능별로 나뉩니다.

처음부터 모두 켜는 구조가 아닙니다. 접근성은 입력 제어의 필수 조건이고, 나머지는 필요한 순간에만 요청하거나 권장합니다.

필수

접근성 서비스

PC 마우스와 키보드로 Android를 제어하고, 스크린샷 요청과 원격 입력을 처리하기 위해 필요합니다.

원격 제어 시작 시

화면 캡처 승인

모바일 화면 원격 제어를 시작할 때 Android 시스템 팝업에서 승인합니다. 보안 화면은 표시되지 않을 수 있습니다.

연결 시

주변 Wi-Fi 기기

Android 13 이상에서 같은 네트워크의 PC를 찾을 때 요청될 수 있습니다. 앱 실행 직후에는 묻지 않습니다.

서비스 시작 시

알림

백그라운드 연결 상태를 표시할 때 사용합니다. 연결 서비스를 켤 때 필요한 경우에만 요청합니다.

밝기 사용 시

시스템 설정 변경

Alt + Shift + ↑ / ↓로 Android 밝기를 조절하려면 첫 사용 시 Android 설정에서 허용해야 합니다.

권장

배터리 제한 예외

화면이 꺼진 뒤 연결이 자주 끊길 때만 확인합니다. 안정성을 높이는 권장 설정입니다.

File Transfer

PC 파일은 전송 화면, 빠른 드롭존, PC-to-PC 드래그로 보냅니다.

전송 화면에서 파일을 고르거나, PC 화면 위 작은 드롭존에 파일을 놓으면 선택된 Android 또는 서브 PC로 전송합니다. PC-to-PC에서는 파일을 잡은 상태로 대상 PC가 배치된 화면 가장자리로 넘긴 뒤 대상 PC에서 놓을 수 있습니다. Android에서 PC로 보낼 때는 Android 전송 탭 또는 공유 메뉴를 사용합니다.

Quick Drop

빠른 드롭존

드롭존 위치와 활성화 상태는 PC 앱에 저장됩니다. 방해될 때는 전역 단축키로 숨기거나 다시 표시합니다.

Ctrl + Alt + D
  • 파일 전송은 선택된 대상에 따라 PC -> Android 또는 PC -> PC 경로를 사용합니다.
  • PC-to-PC 파일 드래그는 대상 서브 PC가 연결되어 있고 배치판에 놓여 있을 때 사용합니다.
  • 파일을 잡은 채 배치된 화면 가장자리로 이동하면 대상 PC로 드래그가 이어지고, 원하는 위치에서 놓으면 전송됩니다.
  • 전송이 시작되지 않으면 Android/PC 연결 상태와 선택된 기기를 먼저 확인합니다.
빠른 드롭존 화면 보기 파일을 놓는 위치와 전송 화면을 확인합니다.

Quick Connect

기기 목록의 번호 단축키로 바로 연결합니다.

PC 앱의 기기 관리 목록에는 첫 9개 기기에 Ctrl + Alt + 1~9 단축키가 표시됩니다. 앱 창을 다시 찾지 않아도 자주 쓰는 Android 또는 서브 PC를 바로 연결할 수 있습니다.

  • Ctrl + Alt + 1은 목록의 첫 번째 기기, Ctrl + Alt + 2는 두 번째 기기를 연결합니다.
  • 이미 연결된 기기의 단축키를 한 번 더 누르면 해당 연결을 끊습니다.
  • 연결되거나 끊기면 PC 화면 오른쪽 아래에 작은 알림이 잠깐 표시됩니다.

Auto Reconnect

다시 보이는 기기는 자동 재연결할 수 있습니다.

자동 재연결을 켜 두면 마지막 기기가 다시 준비되거나 준비된 기기가 하나뿐일 때 PC 앱이 연결을 다시 시도합니다. 마우스가 기기 사이를 오갈 때는 별도 알림을 띄우지 않습니다.

Advanced: PC to PC

다른 Windows PC는 설정, 기기 관리, 배치 순서로 붙입니다.

제어를 시작할 PC는 설정 화면의 PC 역할을 메인 PC로 둡니다. 제어받을 PC는 서브 PC로 실행한 뒤, 메인 PC의 기기 관리 화면에서 대상 PC를 검색하고 상대 PC에 표시된 6자리 코드를 입력해 페어링합니다.

  • 설정 화면에서 PC 역할 버튼이 메인 PC이면 Android와 서브 PC 연결을 시작할 수 있습니다.
  • 기기 관리에서 대상 PC 행이 보이면 상대 PC의 6자리 코드를 입력하고 페어링을 누릅니다.
  • 페어링 뒤 대상 PC 행에 연결됨이 보이면 폰 위치 / 입력 진입 방향 화면에서 배치를 조정합니다.
  • 대상 PC 화면을 보며 조작해야 하면 연결된 대상 PC 행의 원격 제어 아이콘을 누릅니다.

Arrangement

배치 화면에서 실제 진입 방향을 정합니다.

  • 대상 PC를 내 PC 왼쪽에 붙이면 왼쪽 경계로 넘길 때 대상 PC로 제어가 이동합니다.
  • Android도 같은 배치판에 함께 둘 수 있습니다. 대상 PC와 Android가 동시에 연결되어 있으면 각각 따로 배치합니다.
  • 사이드바 연결 상태에는 현재 선택된 대상과 진입 방향이 표시됩니다. 방향이 다르면 배치판에서 다시 드래그합니다.

PC Remote Control

서브 PC 화면도 별도 창에서 원격 제어할 수 있습니다.

  • 메인 PC의 기기 관리에서 연결된 서브 PC 행의 원격 제어 아이콘을 누르면 PC 화면 원격 제어 창이 열립니다.
  • 창이 열리면 대상 PC 화면을 보면서 마우스 클릭, 드래그, 휠 스크롤을 바로 보낼 수 있습니다.
  • PC 키보드 입력도 대상 PC로 전달됩니다. 예를 들어 Ctrl + C, Ctrl + V 같은 단축키는 대상 PC에서 실행됩니다.

Quality

품질은 상황에 맞게 선택하고 다시 연결합니다.

  • 빠름은 반응성을 우선하고, 표준은 기본 사용에 맞춘 설정입니다.
  • 고화질은 화면을 더 선명하게 보고 싶을 때 사용합니다. 네트워크 상태가 좋을수록 안정적입니다.
  • 품질을 바꾼 뒤에는 재연결해야 새 설정이 적용됩니다. 창을 닫으면 원격 제어가 중지됩니다.
PC 원격 제어 화면 예시 보기 기기 관리에서 대상 PC를 선택하고, 별도 창에서 서브 PC 화면을 조작하는 흐름입니다.

PC-to-PC Screen Flow

화면에서는 이 순서로 보면 됩니다.

메인 PC는 연결을 시작하는 PC, 대상 PC는 마우스와 키보드를 넘겨 받을 PC입니다. 서브 PC의 페어링 코드는 화면 하단 6자리 숫자를 그대로 입력합니다.

1
설정에서 PC 역할을 확인합니다

제어를 시작하는 쪽은 설정 화면의 PC 역할이 메인 PC여야 합니다. 메인 PC는 Android와 대상 PC 연결을 시작할 수 있습니다.

2
서브 PC의 페어링 코드를 확인합니다

제어받을 PC에서도 OneMouse를 실행하고, 사이드바 하단의 6자리 페어링 코드를 확인합니다. 예시는 101884처럼 보이는 숫자 코드입니다.

3
기기 관리에서 대상 PC를 페어링합니다

메인 PC의 기기 관리 화면에서 대상 PC 행을 찾고, 상대 PC 6자리 코드 입력칸에 서브 PC 코드를 넣은 뒤 페어링을 누릅니다.

4
연결됨과 선택됨 상태를 확인합니다

대상 PC 행이 연결됨으로 바뀌고 선택됨 배지가 붙으면 현재 마우스 진입 대상입니다. Android와 대상 PC가 함께 보이면 선택된 기기를 기준으로 동작합니다.

5
폰 위치 / 입력 진입 방향에서 배치합니다

대상 PC 사각형을 내 PC 왼쪽, 오른쪽, 위, 아래 중 실제 놓인 방향으로 드래그합니다. Android도 같은 배치판에 함께 둘 수 있습니다.

PC-to-PC 화면 예시 보기 메인 PC, 서브 PC, 페어링, 연결됨, 배치 화면을 순서대로 확인합니다.

Keyboard Shortcuts

PC 앱 전역 단축키와 Android 제어 단축키입니다.

빠른 연결과 드롭존 단축키는 PC에서 바로 동작합니다. Android 제어 단축키는 마우스가 Android로 넘어간 상태에서 동작합니다.

단축키 동작 메모
Ctrl + Alt + 1~9 기기 목록의 첫 9개 기기를 바로 연결합니다. 이미 연결된 기기 번호를 다시 누르면 해당 연결을 끊습니다.
Ctrl + Alt + D 빠른 드롭존을 숨기거나 다시 표시합니다. 파일을 드롭존에 놓으면 선택된 Android 또는 서브 PC로 전송합니다.
Enter 검색창에서는 검색/완료를 실행합니다. 일반 줄바꿈 대신 Android IME의 검색/완료 액션을 우선 보냅니다.
Shift + Enter 입력창에 줄바꿈을 넣습니다. 채팅이나 긴 문장 입력에서 줄을 나눌 때 사용합니다.
Ctrl + A 현재 입력창의 텍스트를 전체 선택합니다. 앱이 선택 범위를 직접 노출하지 않는 경우 OneMouse가 선택 상태를 보조합니다.
Ctrl + C / X / V 복사, 잘라내기, 붙여넣기를 실행합니다. 복사와 잘라내기는 가능하면 PC 클립보드로도 바로 동기화합니다.
Shift + 방향키 텍스트 선택 범위를 확장합니다. 일부 앱은 Android 접근성 선택 동작을 제한할 수 있습니다.
Alt + ↑ / ↓ Android 미디어 볼륨을 조절합니다. 입력창 포커스가 있어도 우선 실행하며, 누르고 있으면 반복 조절됩니다.
Alt + Shift + ↑ / ↓ Android 화면 밝기를 조절합니다. 처음 사용할 때 Android의 시스템 설정 변경 권한을 허용해야 합니다.
Print Screen Android 화면을 캡처해 PC 기본 이미지 뷰어로 엽니다. Android 11 이상에서 접근성 스크린샷 기능을 사용합니다. 보안 화면은 캡처되지 않을 수 있습니다.
Ctrl + Alt + S Print Screen 대신 모바일 스크린샷을 실행합니다. 노트북이나 원격 키보드에서 Print Screen 키를 쓰기 어려울 때 사용합니다.
Alt + Tab Android 최근 앱 화면을 열고 앱을 전환합니다. Alt를 누른 채 Tab을 반복하면 다음 앱으로 이동하고, Alt를 떼면 선택합니다.
Ctrl + Alt + Backspace PC로 입력 제어를 강제 복귀합니다. 커서가 Android 쪽에 남아 있을 때 PC 입력으로 되돌리고, PC에 마우스 복귀 알림을 표시합니다.

Daily Use

연결 후 자주 쓰는 흐름

  • PC에서 현재 제어할 Android 또는 서브 PC를 선택하고 위치를 조정합니다.
  • 모바일 검색창, 로그인 입력, 메시지 작성은 PC 키보드로 처리합니다.
  • PC 파일은 전송 화면에서 고르거나 화면 위 빠른 드롭존에 놓아 선택된 Android 또는 서브 PC로 보냅니다.
  • 서브 PC로 보낼 때는 파일을 잡은 채 배치된 화면 가장자리로 넘겨 대상 PC에서 놓을 수도 있습니다.
  • 텍스트를 꾹 눌러 선택한 뒤 OneMouse to PC를 누르면 PC 클립보드로 바로 보냅니다.
  • 모바일 화면을 계속 보며 조작해야 하면 기기 관리의 원격 제어 아이콘으로 모바일 화면 원격 제어를 시작합니다.
  • 한 장만 확인하면 Print Screen 또는 Ctrl + Alt + S로 스크린샷을 찍어 PC에서 확인합니다.

Recovery

커서가 돌아오지 않을 때

Android로 넘어간 커서가 PC로 복귀하지 않으면 PC에서 Ctrl + Alt + Backspace를 누릅니다. 이 단축키는 마우스 입력을 PC로 되돌리고 Android에 복귀 신호를 다시 보낸 뒤 PC에 복귀 알림을 표시합니다.

모바일 스크린샷은 PC에서 바로 확인합니다.

Android 화면을 캡처하면 PC로 전송되고 Windows 기본 이미지 뷰어로 열립니다. 보안 정책이 걸린 화면이나 Android 접근성 스크린샷이 허용되지 않는 상태에서는 실패할 수 있습니다. 계속 보며 조작할 때는 모바일 화면 원격 제어를 사용하고, 한 장만 확인할 때는 스크린샷 단축키를 사용합니다.

When It Fails

실패할 때 확인할 것

  • 앱 업데이트 후 OneMouse 접근성 서비스를 껐다가 다시 켭니다.
  • 은행, DRM, 보안 키보드처럼 캡처 금지 화면인지 확인합니다.
  • 너무 빠르게 연속 캡처했다면 잠시 기다린 뒤 다시 시도합니다.
  • PC와 Android 연결이 유지되어 있는지 확인합니다.
