오픈소스 라이선스

OneMouse Android 앱은 아래 라이브러리를 사용합니다. 각 라이브러리의 저작권과 라이선스 조건은 원 배포처의 고지를 따릅니다.

최종 업데이트: 2026년 5월 9일

배포 앱에 포함되는 주요 라이브러리

  • AndroidX Core KTX 1.13.1
    용도: Android 호환성 유틸리티와 Kotlin 확장 기능
    라이선스: Apache License 2.0
    링크: AndroidX, Apache License 2.0
  • AndroidX AppCompat 1.6.1
    용도: 하위 Android 버전 호환 UI 구성
    라이선스: Apache License 2.0
    링크: AndroidX AppCompat, Apache License 2.0
  • Google Play Billing Library 8.3.0
    용도: 향후 PRO 인앱결제와 구매 복원 처리
    제공: Google
    링크: Google Play Billing
  • ZXing Android Embedded 4.3.0
    용도: PC 화면의 QR 코드 스캔
    라이선스: Apache License 2.0
    링크: zxing-android-embedded, ZXing, Apache License 2.0

빌드와 테스트 전용

다음 의존성은 테스트 또는 빌드 과정에 사용되며, 일반 사용자에게 배포되는 앱 기능의 런타임 라이브러리로 직접 고지하지 않습니다.

  • JUnit 4.13.2: 테스트 코드 실행
  • org.json 20240303: 테스트 코드의 JSON 처리
  • Android Gradle Plugin, Kotlin Gradle Plugin, Gradle wrapper: 빌드 도구

고지 기준

이 페이지는 OneMouse가 사용하는 외부 라이브러리를 사용자에게 알리기 위한 요약입니다. 라이선스 원문, 저작권 표시, 변경 조건은 각 라이브러리의 공식 저장소와 배포 문서가 우선합니다.
