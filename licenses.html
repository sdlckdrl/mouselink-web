오픈소스 라이선스

OneMouse는 PC 앱, Android 앱, QR 스캔, 원격 제어 스트리밍, 결제, 광고 기능을 위해 오픈소스 라이브러리와 서드파티 SDK를 사용합니다. 각 구성요소의 저작권과 라이선스 조건은 원 배포처의 고지를 따릅니다.

최종 업데이트: 2026년 6월 19일

Android 앱

  • AndroidX Core KTX 1.13.1
    용도: Android 호환성 유틸리티와 Kotlin 확장 기능
    라이선스: Apache License 2.0
  • AndroidX Activity KTX 1.9.3
    용도: Android Activity API와 Kotlin 확장 기능
    라이선스: Apache License 2.0
  • AndroidX AppCompat 1.6.1
    용도: 하위 Android 버전 호환 UI 구성
    라이선스: Apache License 2.0
  • Kotlin standard library
    용도: Kotlin 런타임 지원
    라이선스: Apache License 2.0
  • ZXing Android Embedded 4.3.0 / ZXing Core
    용도: PC 화면의 QR 코드 스캔
    라이선스: Apache License 2.0
  • Google Play Billing Library 8.3.0
    용도: Pro 인앱결제와 구매 복원
    조건: Google SDK 약관
  • Google Mobile Ads SDK 25.0.0
    용도: Free 모드 배너 광고
    조건: Google SDK 약관
  • Google Play App Update SDK 2.1.0
    용도: Android 앱 업데이트 확인
    조건: Google SDK 약관
  • WebRTC Android SDK 144.7559.05
    용도: 모바일 화면 원격 제어의 영상 스트리밍
    라이선스: BSD 3-Clause License

빌드와 테스트 전용

Android Gradle Plugin, Kotlin Gradle Plugin, Gradle wrapper, JUnit, org.json, Electron Builder 등은 빌드나 테스트 과정에 사용됩니다. 배포 파일에 포함되는 런타임 구성요소는 위 목록을 기준으로 고지합니다.

고지 기준

이 페이지는 OneMouse 배포판에 포함되거나 주요 기능 제공에 사용되는 라이브러리와 SDK를 사용자에게 알리기 위한 고지입니다. 아래에 각 라이선스의 전문(또는 공식 원문 링크)과 저작권 표시를 포함합니다. LGPL 구성요소의 경우, 해당 라이선스 사본은 요청 시 제공하며 공식 원문 링크를 함께 안내합니다.
