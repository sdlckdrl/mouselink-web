Help

Quickly narrow down permission, discovery, and return issues.

It is faster to split the cause into app power, permissions, network discovery, pairing, and placement. Check the items below in order and you can narrow down most blockers.

Power

Check the app power

If the power in the top-right is off, background activity and input receiving stop. Turn the power on first when you start a connection.

Permissions

Permissions come first

Android input control requires the Accessibility service to be on. If the permission is off, real input does not go through even when the PC is visible.

Battery

Check battery limits

If the connection drops often after the screen turns off, check the battery limit exception in Android settings. You can jump there directly from the battery limit item in the app settings.

Pairing

QR or 6-digit code

Scan the QR at the bottom of the PC or enter the 6-digit code. It is normal for the connect button to stay disabled before pairing.

Device Search

Check network discovery

On company networks, guest Wi-Fi, and some router settings, device discovery may be blocked. First check that you are under the same router.

Firewall

Check the firewall and network

If the PC is not visible, check whether you are on the same Wi-Fi or under the same router, and whether a guest Wi-Fi or company network policy is blocking it.

Layout

When the placement is off

After an Android orientation change, the screen size and position are synced again. If it still feels off, save the phone position again.

Return

Cursor return shortcut

If the cursor does not come back to the PC, press Ctrl + Alt + Backspace to return mouse input to the PC. When the return runs, a small notification appears on the PC.

Troubleshooting

Troubleshooting order

Splitting out which step you are stuck at first makes the fix much faster.

1
Check the app and permission state

Confirm the PC app is on, and that the Android app power and Accessibility permission are on.

2
Check whether the PC is visible

Confirm that device search works under the same Wi-Fi or the same router, then run device search again.

3
Check the pairing state

If the PC list shows pairing required, use the QR or 6-digit code again. If it is already tangled, unpair and connect again.

4
Check input and placement

On the PC, check the selected device, the Android position, the entry direction, and the sync state after a screen orientation change.

5
Use the recovery shortcut

If the cursor feels trapped on the Android side, force PC input back with Ctrl + Alt + Backspace. On a forced return, a return notification appears on the PC.

Transfer

When files and clipboard do not work

  • Check which Android device or Sub PC is selected on the PC. PC -> Android/PC files go to the selected device.
  • Press OneMouse to PC in the text selection menu, or choose OneMouse from the Android share menu, to send it to the default PC.
  • On a Sub PC, the paired Main PC must appear as the connection target before PC -> PC file transfer is possible.
  • If OneMouse to PC does not appear, turn on One Mouse in Manage apps under the More menu. This setting is usually only needed once at the start.
  • Large files are affected by the network state, so first check that the connection is stable.
  • The clipboard reuses the file transfer TCP channel, so the pairing and session key state matters.

FAQ

Frequently asked questions

The answers have been reorganized around the latest connection structure.

Does it go through an intermediate server?
The default structure is a direct connection between the PC and Android under the same Wi-Fi or the same router.
What do I do if the PC is not discovered?
Make sure the PC and Android are on the same Wi-Fi or under the same router, and check that you are not in an environment where devices cannot see each other, such as guest Wi-Fi or a company network.
I paired over Bluetooth, but the PC cannot find my phone.
Pair the phone once in Windows Bluetooth settings — that is all you need. There is no COM port or serial port to set up. After pairing, launching the PC app connects automatically. If the PC cannot find the phone, make sure Bluetooth mode (Beta) is on in the OneMouse Settings tab on the phone and that the phone is paired in Windows, then wait a moment. If it still fails, remove the device in Windows and pair again.
What is the difference between scanning the QR and the 6-digit code?
Both are ways to start the same pairing. The QR reduces typing, and the 6-digit code can be used when the camera is hard to use.
Can I connect several Android devices at once?
Yes. You can discover, select, connect, and place multiple devices on the PC.
Can I control another Windows PC too?
Yes. Search for the target PC on the Main PC, pair it with the 6-digit code, then place the target PC on the Phone position / input direction screen.
What do I do if the cursor does not return from Android to the PC?
Press Ctrl + Alt + Backspace on the PC. It is the recovery shortcut that returns mouse input to the PC.
Can I send text or files from Android to the PC?
You can send text and files to the default PC by choosing OneMouse from the Android share menu or by using clipboard sync.
