Guide

From first connection to PC-to-PC, in order.

Follow the core Android setup first, then use file transfer, keyboard shortcuts, and PC-to-PC as needed.

1
Run the PC app

Open OneMouse on Windows. The PC app shows a 6-digit pairing code and QR code at the bottom.

2
Turn on Android power and Accessibility

The power button must be on for discovery and input receivers. Accessibility is required to control Android input.

3
Search for the PC

Search while both devices are on the same Wi-Fi or router. If the PC is not visible, check the network and run device search again.

4
Pair with QR or code

Scan the QR code or enter the 6-digit code from the PC app. Unpaired devices remain in a pairing-required state.

5
Place the phone on a PC edge

Choose left, right, top, or bottom based on where you want the cursor to enter Android.

6
Use input, files, and clipboard

Move the PC mouse across the configured edge. Send PC files from the transfer screen or the Quick Drop zone to Android or a paired Sub PC, and send selected Android text back to the PC clipboard.

App Screens

Where to look in the app.

For first setup, check the PC code and QR, the Android pairing button, and the PC placement screen in that order.

File Transfer

Send PC files from the transfer screen or Quick Drop.

Choose files from the transfer screen, or drop them onto the small floating drop zone on the PC screen. Files go to the selected Android device or paired Sub PC.

Quick Drop

Toggle the drop zone

Use the global shortcut when the drop zone is in the way or when you need it again.

Ctrl + Alt + D
  • The PC app saves the drop zone position and enabled state.
  • Quick Drop uses the PC-to-Android or PC-to-PC file transfer path based on the selected target.
  • If transfer does not start, check the Android/PC connection and selected device first.

Advanced: PC to PC

Set the role, pair the target PC, then place it.

Keep the controlling computer in Main PC role. Run the other computer as a Sub PC, search for it from Device management, enter the 6-digit code shown on that PC, and pair it.

  • If the PC role button says Main PC, this computer can start Android and Sub PC connections.
  • When the target PC row appears in Device management, enter the other PC's 6-digit code and press Pair.
  • After the row shows Connected, open Phone position / input direction to place the target PC.

Arrangement

Arrangement decides the real entry direction.

  • Place the target PC on the left of this PC to enter it from the left monitor edge.
  • Android devices can stay on the same arrangement board. Place each connected target separately.
  • The sidebar shows the selected target and current entry direction. If it is wrong, drag the target again.

Need help?

Use Help when setup stops at a specific step.

Network discovery, pairing, input handoff, cursor return, and transfer issues are separated in the Help page.
