Use Cases

Keep working from your PC across phone and PC screens.

You do not need to understand network details first. Start with the tasks you already do every day: control, type, copy, and send files.

Phone Control

Use Android like a nearby screen

Push the mouse past a PC screen edge and continue clicking, scrolling, and controlling Android.

Typing

Type long text with your PC keyboard

Use your PC keyboard for searches, messages, forms, and any Android field where typing on glass is slow.

Files

Drag files into Quick Drop

Pick up a file on your PC and drop it into the small send zone to send it to Android or another paired PC.

Clipboard

Move selected text faster

Send selected Android text back to the PC or use clipboard features when you repeat the same copy flow often.

PC to PC

Control another Windows PC

Place a paired Windows PC on a monitor edge, move the cursor across, and send files between PCs.

Bluetooth

Connect directly over Bluetooth — no Wi‑Fi

Even without a router or internet, connect to your phone and other PCs over Bluetooth to share mouse, keyboard, clipboard, and files. You can also connect several devices at once.

Quick Connect

Connect from the device list

The PC app shows Ctrl + Alt + 1-9 shortcuts for devices. Press the same shortcut again to disconnect, with small desktop toasts for connection and return status.

Control

Turn it off when you are done

The Android power button stops connection waiting and input receiving when you are not using OneMouse.

Return

Recover the cursor quickly

If the cursor feels stuck on a target device, press Ctrl + Alt + Backspace on the PC to return input.

Pro

Expand repeated daily use

Upgrade when you want no ads, multiple saved PCs, and fewer file or clipboard limits.

Setup Flow

First setup takes four clear steps.

Follow the PC and Android screens. The app handles the network details for you.

01

Open the PC app

Run OneMouse on Windows and check the QR code or 6-digit code at the bottom.

02

Enable Android permission

Turn on Accessibility so Android can receive mouse and keyboard input from the PC.

03

Pair with QR or code

Find the PC on the same Wi-Fi and confirm it with QR scan or the 6-digit code.

04

Place and use

Place Android or another PC on a screen edge, then move the cursor across that edge.

Before You Connect

Check these three things first.

OneMouse connects devices you own or control. Most setup issues come down to network visibility, Android permission, or pairing state.

Same Wi-Fi or router Your PC and Android device need to be able to see each other on the network.
Android Accessibility Android needs Accessibility permission before it can receive PC input.
QR or 6-digit code Only paired devices can use mouse, keyboard, file, and clipboard features.
