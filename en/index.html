Use Cases

Keep working from your PC across phone and PC screens.

You do not need to understand network details first. Start with the tasks you already do every day: control, type, copy, and send files.

Phone Control

Use Android like a nearby screen

Push the mouse past a PC screen edge and continue clicking, scrolling, and controlling Android.

Typing

Type long text with your PC keyboard

Use your PC keyboard for searches, messages, forms, and any Android field where typing on glass is slow.

Files & Clipboard

Send files and text instantly

Drag a file to the screen edge to send it to the selected device, and your text and clipboard stay synced on both sides.

PC to PC

Control another Windows PC

Place a paired Windows PC on a monitor edge, move the cursor across, and send files between PCs.

Bluetooth

Setup Flow

First setup takes four clear steps.

Follow the PC and Android screens. The app handles the network details for you.

01

Open the PC app

Run OneMouse on Windows and check the QR code or 6-digit code at the bottom.

02

Enable Android permission

Turn on Accessibility so Android can receive mouse and keyboard input from the PC.

03

Pair with QR or code

Find the PC on the same Wi-Fi and confirm it with QR scan or the 6-digit code.

04

Place and use

Place Android or another PC on a screen edge, then move the cursor across that edge.
