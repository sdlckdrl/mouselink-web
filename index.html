Windows + Android 연결 워크플로

마우스와 키보드로
PC와 모바일을
끊김 없이 넘나듭니다

MouseLink는 PC 입력을 Android로 자연스럽게 넘기고, 페어링과 파일 전송까지 한 화면 안에서 이어 주는 데스크톱-모바일 연결 앱입니다. 작업 흐름을 멈추지 않고 메시지 확인, 검색, 텍스트 입력, 파일 전달을 바로 처리할 수 있습니다.

다운로드 시작 가이드 보기
마우스 브리지 키보드 입력 파일 전송 6자리 코드 페어링

PC에서 하던 흐름 그대로 모바일까지 이어집니다

마우스를 가장자리로 넘겨 제어를 이동하고, PC 키보드로 입력하고, 필요한 파일은 바로 보내는 식으로 데스크톱 작업의 리듬을 유지할 수 있습니다.

01

부드러운 마우스 브리지

모니터 가장자리로 커서를 밀면 Android 화면으로 진입합니다. PC에서 쓰던 포인터 흐름이 이어져 별도 리모컨 앱처럼 느껴지지 않습니다.

02

키보드 입력 그대로 전달

검색창, 메신저, 로그인 폼처럼 긴 입력이 필요한 상황에서 PC 키보드로 바로 입력할 수 있어 모바일 타이핑 부담을 줄여 줍니다.

03

빠른 파일 전송

작업 중인 문서, 이미지, APK, 테스트 파일 등을 PC에서 Android로 바로 보내 확인할 수 있습니다. 테스트 빌드 검증에도 잘 맞습니다.

04

코드 기반 페어링

처음 연결할 때 6자리 코드를 확인해 같은 네트워크 안의 다른 기기가 임의로 붙지 않도록 기본적인 보호 절차를 거칩니다.

05

여러 모바일 연결 지원

여러 Android 기기를 페어링해 두고 PC 가장자리 배치에 따라 전환할 수 있습니다. 테스트용 폰을 여러 대 운용할 때 특히 편합니다.

06

로컬 네트워크 중심 구조

같은 Wi-Fi 환경에서 PC와 Android가 직접 통신하는 흐름을 기본으로 하므로, 사내 테스트나 개인 작업 환경에서 빠르게 붙여 쓰기 좋습니다.

처음 연결하는 방법

한 번 페어링해 두면 이후에는 같은 네트워크에서 더 빠르게 다시 연결할 수 있습니다.

PC 앱 실행 Windows에서 MouseLink를 실행하고 장치 검색이 시작된 상태를 유지합니다.
Android 앱 열기 모바일에서 앱을 열고 검색 대기 상태를 확인합니다. 필요 권한이 있으면 먼저 허용합니다.
6자리 코드 확인 PC에 표시된 6자리 코드를 모바일에 입력해 같은 기기인지 확인하며 페어링합니다.
배치 후 바로 사용 연결이 끝나면 가장자리 방향과 기기 위치를 맞춘 뒤 마우스와 키보드를 바로 사용할 수 있습니다.

빠른 시작 체크리스트

  • PC와 Android가 같은 Wi-Fi 또는 같은 로컬 네트워크에 연결되어 있는지 확인
  • PC 앱이 실행 중인지 확인
  • Android 앱에서 검색 대기 또는 연결 가능 상태인지 확인
  • 처음 연결이라면 6자리 페어링 코드를 입력했는지 확인
  • 마우스 진입 방향과 폰 배치가 현재 모니터 배치와 맞는지 확인

이런 상황에서 잘 맞습니다

  • 앱 테스트 중 여러 Android 기기에서 반복 입력이 필요할 때
  • 문서, 링크, 이미지 파일을 모바일로 빠르게 보내 확인할 때
  • PC 키보드로 모바일 검색이나 로그인 정보를 입력하고 싶을 때
  • 메신저 답장이나 인증 코드 입력을 작업 흐름 안에서 끝내고 싶을 때

도움말과 사용 팁

설치 직후 자주 헷갈리는 권한, 연결 조건, 화면 배치, 파일 전송 흐름을 한 번에 정리했습니다.

권한 안내

  • Android 쪽에서 입력 제어를 받으려면 접근성 또는 입력 관련 권한이 필요할 수 있습니다.
  • 파일 수신 기능을 쓰려면 저장소 또는 파일 접근 권한이 필요할 수 있습니다.
  • 권한이 꺼져 있으면 기기 검색은 되지만 실제 입력 전달이 동작하지 않을 수 있습니다.

연결이 잘 안 될 때

  • PC와 Android가 서로 다른 게스트 Wi-Fi나 분리된 네트워크에 있지 않은지 확인합니다.
  • 회사 네트워크나 공용 Wi-Fi에서는 로컬 브로드캐스트가 제한될 수 있습니다.
  • Windows 방화벽 또는 백신이 앱의 네트워크 통신을 막고 있지 않은지 확인합니다.
  • Android 앱을 백그라운드에서 강제 종료했다면 다시 열어 검색 상태를 유지합니다.

마우스 진입 위치 맞추기

  • PC 모니터의 왼쪽, 오른쪽, 위, 아래 중 실제로 폰이 놓인 방향과 동일하게 설정합니다.
  • 여러 대를 연결했다면 각 폰의 배치가 겹치지 않도록 나누어 두는 편이 좋습니다.
  • 진입이 어색하면 폰 위치를 조금 조정한 뒤 다시 가장자리 진입을 시험해 봅니다.

파일 전송 팁

  • 테스트 중인 APK, 스크린샷, 문서 파일처럼 자주 오가는 파일 전달에 적합합니다.
  • 전송 전 대상 Android가 연결 상태인지 먼저 확인하면 실패를 줄일 수 있습니다.
  • 대용량 파일은 네트워크 품질에 따라 시간이 더 걸릴 수 있으니 전송 완료 상태를 확인합니다.

문제 해결 가이드

연결이 보이지 않거나 입력이 안 넘어갈 때는 아래 순서로 점검하면 대부분 바로 원인을 찾을 수 있습니다.

1. 네트워크 확인 두 기기가 같은 공유기 아래에 있는지 먼저 확인합니다.
2. 앱 상태 확인 PC 앱과 Android 앱이 모두 열려 있고 검색 또는 연결 대기 상태인지 확인합니다.
3. 권한 확인 Android 접근성, 입력, 파일 관련 권한이 꺼져 있지 않은지 확인합니다.
4. 재시도 페어링을 다시 시도하거나 앱을 재실행해 세션을 새로 잡습니다.

자주 묻는 질문

설치 전후로 많이 확인하는 질문을 빠르게 정리해 두었습니다.

인터넷 서버를 거치나요?
기본 구조는 같은 로컬 네트워크 안에서 PC와 Android가 직접 통신하는 방식입니다. 따라서 같은 공간의 작업 환경에서 빠르게 붙여 쓰기 좋습니다.
왜 6자리 코드가 필요한가요?
처음 연결 시 사용자가 의도한 기기끼리 연결하는지 확인하기 위한 절차입니다. 같은 네트워크 안의 다른 기기가 임의로 붙는 상황을 줄여 줍니다.
모바일 쪽 앱도 계속 켜 두어야 하나요?
입력을 받는 동안에는 Android 앱이 연결을 유지할 수 있어야 합니다. 일부 기기에서는 배터리 최적화 설정 때문에 백그라운드 유지 정책도 함께 확인하는 편이 좋습니다.
여러 대의 Android를 함께 쓸 수 있나요?
여러 기기를 페어링하고 연결해 둔 뒤, PC 가장자리 배치에 따라 각각 다른 모바일로 진입하는 구성이 가능합니다. 실제 사용성은 배치 설정에 크게 영향을 받습니다.
파일 전송은 어떤 방향으로 되나요?
현재 구조상 PC에서 Android로 보내는 흐름을 중심으로 안내하는 것이 가장 자연스럽습니다. 앱 구현 상태에 따라 Android에서 PC로의 공유 흐름도 추가될 수 있습니다.
키보드 입력이 이상하게 들어가면 어떻게 하나요?
Android 입력기 상태, 언어 전환 상태, 접근성 권한 여부를 먼저 확인해 보세요. 앱을 재연결하면 입력 세션이 정상화되는 경우도 많습니다.
